# Syntracker Fx (GoldFX Journal)

Trading Journal premium diseñado bajo las reglas del **GoldFX Discipline Score**.

## Estructura del Proyecto

* `index.html`: Aplicación web principal (versión para servidor/Vercel).
* `sfx-journal-v3 (1).html`: Versión de escritorio (local).
* `logo.png`: Logotipo oficial del proyecto.
* `gold_bull.png`: Asset de fondo (toro dorado).
* `dev-tools/`: Scripts de automatización y diseño utilizados durante el desarrollo.
* `.vercel/`: Configuración de despliegue para Vercel.

## Características

1. **Alineación de Marca**: Diseño premium en colores Black & Gold (#ffcd1b) y tipografía elegante Inter.
2. **Discipline Score**: Sistema estricto de auditoría para evaluar la consistencia, gestión de riesgo y control emocional de cada operación.
3. **Cerebro GoldFX**: Módulo analítico inteligente que analiza tus rachas y emociones dominantes por periodo de tiempo.
4. **Calendario y Estadísticas**: Visualización detallada de rendimientos diarios, semanales y mensuales.
5. **Aislamiento de Errores**: Robustez optimizada en las tarjetas de cuentas y filtros de inacción.

## Despliegue

Este proyecto está configurado para desplegarse en Vercel. Si deseas realizar cambios, puedes usar:
```bash
npx vercel --prod
```
